Keywords are reserved words for a special purpose.

For example the word *function* is reserved to declare a function, and cannot be used for any other purpose.

## Example

The following declarations will be invalid:

### Invalid Example 1

```
Function Function( )
Begin
    return;
End
```

### Invalid Example 2

```
int function;
float debug;
```
